# My Tech Blog
My tech blog will be built using Next.js and will primarily feature content created with Markdown files.

## Get Started
1. Clone the repo
```
git clone https://github.com/deepakmardii/blog.git
```
2. Go to the folder
```
cd blog
```
3. Start the server
```
npm run dev
```
